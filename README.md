## 配置 Travis CI

* 添加 .travis.yml
```yaml
language: go
```

* 使用 GitHub 账号登陆 [Travis CI](https://travis-ci.org/)，获取 gopl 组织访问权限

* [添加仓库](https://travis-ci.org/profile/gopl)

	将 gopl/ci 设置为打开状态

* 设置 Build Status

	访问 [Travis 仓库](https://travis-ci.org/gopl/ci)，点击图标，选择 Markdown，复制、粘贴到 README.md 顶部
